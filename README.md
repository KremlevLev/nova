# Nova: Локальный ИИ-ассистент для Windows

Nova — это компактный и оптимизированный под работу на процессоре (CPU) голосовой ассистент, который управляет операционной системой Windows, ставит напоминания, сохраняет заметки и ищет информацию в сети.

---

## 🛠️ Особенности архитектуры

* **Слух (STT):** Локальный захват звука с микрофона с помощью кастомного программного RMS VAD (детектора тишины) + мгновенное распознавание через сверхбыстрое **Groq Whisper API** (`whisper-large-v3-turbo`).
* **Голос (TTS):** Оптимизированный не-авторегрессионный синтез **Silero TTS v5** (голос `baya` — мягкий и естественный женский тембр). Звук генерируется в один проход напрямую в ОЗУ без использования тяжелых трансформеров.
* **Локальная память (RAG):** Хранилище векторов на чистом `NumPy` по косинусному сходству на базе эмбеддера `FastEmbed` (`intfloat/multilingual-e5-large`).
* **Планировщик задач:** Асинхронная фоновая корутина для проверки и озвучивания отложенных напоминаний.
* **Быстрые обходы (Bypasses):** Мгновенное (до 10мс) выполнение базовых команд по регулярным выражениям и прямой запуск/закрытие программ через нечеткий поиск ярлыков Windows (`.lnk`) в обход LLM.

---

## 📋 Системные требования и зависимости

* **ОС:** Windows 10 / 11
* **Язык программирования:** Python 3.14 (акцент на чистый код и минимум C++ компиляторов при установке)

## 🚀 Пошаговое развертывание

### Шаг 1: Подготовка окружения
Склонируйте репозиторий проекта и перейдите в его корневую директорию. Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

### Шаг 2: Установка голосовой модели
Для работы синтеза речи необходимо скачать локальную модель Silero TTS v5. Для этого запустите скрипт автоустановки:

```bash
python silero_install.py
```
*Данный скрипт автоматически скачает модель весом ~150 МБ в директорию `data/v5_ru.pt` и выполнит первый запуск для кэширования.*

### Шаг 3: Настройка ключей доступа (.env)
Создайте файл `.env` в корневой папке проекта со следующим содержимым:

```env
# Обязательный ключ для работы STT (распознавания) и LLM (мозга)
GROQ_API_KEY=gsk_your_groq_api_key_here

# Рекомендуемый ключ для поиска актуальной информации в интернете
TAVILY_API_KEY=tvly-your_tavily_key_here
```

> ⚠️ **Важно:** 
> * **`GROQ_API_KEY`** является обязательным для запуска проекта. Без него ассистент не сможет обрабатывать запросы и распознавать голос.
> * **`TAVILY_API_KEY`** необходим для функции веб-поиска (инструмент `search_web_tavily`). Если ключ отсутствует, ассистент выдаст ошибку при попытке поиска в реальном времени.

---

## 💻 Запуск ассистента

После успешной настройки окружения запустите главный файл оркестратора:

```bash
python main.py
```

При первом старте Nova поприветствует вас голосом, после чего перейдет в режим ожидания команд. Для завершения работы скажите ассистенту: *«отключайся»* или *«усни»*.

## 🗺️ Планы по развитию проекта (Roadmap)

1. **Канал активации (Wake word / Hotkey):** Добавление в асинхронный цикл горячих клавиш (например, двойное нажатие `Caps Lock` или комбинация `Win + Shift + Z`), прерывающих ожидание и мгновенно активирующих запись. Это снизит нагрузку на CPU от постоянного мониторинга тишины.
2. **Графический интерфейс:** Реализация прозрачного оверлея `modules/ui/overlay.py` на базе `CustomTkinter` с пульсирующей RMS-волной во время записи голоса.
3. **Мультимодальность:** Добавление функции `take_screenshot()` для отправки скриншотов в модели `llama-3.2-11b-vision` от Groq, чтобы Nova могла отвечать на вопросы о происходящем на экране.
4. **Интеграция фолбеков:** Настройка бесшовного переключения на модели через OpenRouter на случай сбоев или превышения лимитов Groq.
